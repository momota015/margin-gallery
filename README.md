# Margin - 余白 -

イラストレーションと写真をテーマにした架空の展示会サイトです。

「余白」というコンセプトをもとに、作品だけでなく閲覧者の想像力を引き出す空間づくりを意識して制作しました。落ち着いたデザインや余白を活かしたレイアウトによって、展示会の世界観が伝わるサイトを目指しました。

## Features

* 展示会をイメージしたランディングページ
* Artistページの実装
* Contactページの実装
* ページ内リンクによるスムーズな遷移
* Google Mapsの埋め込み
* レスポンシブ対応
* 余白を意識したレイアウト設計

## Challenges

* 展示会の静かな雰囲気を表現するため、余白を活かしたレイアウトを設計
* 複数ページ間の導線を意識してナビゲーションを構築
* 写真や作品が引き立つよう文字サイズや余白を細かく調整
* PCとスマートフォンの両方で見やすいレスポンシブデザインを実装
