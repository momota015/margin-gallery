# Margin - 余白 - 🎨
イラストレーションと写真をテーマにした架空の展示会サイトです。
「余白」というコンセプトをもとに、作品だけでなく閲覧者の想像力を引き出す空間づくりを意識し、展示会の世界観が伝わるサイトを目指しました。

## Features
- 展示会をイメージしたトップページ
- アーティスト紹介ページ(個別プロフィール表示)
- お問い合わせフォームページ
- トップページからアーティスト個別紹介へのアンカーリンク
- Google Mapsの埋め込み
- PC・スマホ両対応のレスポンシブ対応

## Challenges
- 「余白」というコンセプトの表現として、文字サイズ・行間・要素間の余白を細かく調整し、展示会の静かな雰囲気が伝わるレイアウトを設計
- 展示会らしい上品な印象を出すため、ヒーローセクションの両サイドに`writing-mode`を使った縦書きテキストを配置し、左右対称で落ち着いた構成を実装
- CSS GridとFlexboxの使い分けとして、要素を整列させたい箇所(アーティスト一覧・詳細)ではGridを、柔軟な配置が必要な箇所(ヒーロー・アクセス)ではFlexboxを使い分けて実装
- スマホ表示での見え方を考慮し、縦書きを横書きに切り替えたり、4カラムを2カラムに変更したりと、レイアウト構造自体を組み替えるレスポンシブ対応を実装
- 複数ページ間の導線として、トップページからアーティスト個別紹介ページのアンカーへ遷移する仕組みを設計

## What I Learned
- CSS GridとFlexboxの特性の違いと、用途に応じた使い分け
- `writing-mode`など、CSSだけで実現できる表現の幅
- 余白や文字サイズといった細部の調整が、サイト全体の印象を大きく左右すること
- レスポンシブ対応では「縮める」だけでなく「組み替える」発想が必要なこと
